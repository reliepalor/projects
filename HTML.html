Here's what I learn in html so far:

    DAY 1
      ATTRIBUTES
    • href - Specifies the hyperlink reference or the URL of the linked resource. For < a > elements, it defines the destination of the link; for < link > elements, it defines the external resource being linked (e.g., a stylesheet).
    • src - Specifies the source URL or file path of the embedded content. For images, it points to the image file; for scripts, it points to the script file; for iframes, it points to the content to be embedded.
    • title - Provides additional information about the element when the user hovers over it. It is a tooltip that displays extra text.
      PARAGRAPHS
    • < p > - defines a paragraph.
    • < hr > - thematic change of content.
    • < br > - Insert a single break
    • < pre > - Defines pre-formatted text.
      STYLES
    • background - color - change the background color.
    • color - change the text color.
    • font - family - change the font of the text.
    • font - size - change the size of the text.
    • text - align - to position the text.
